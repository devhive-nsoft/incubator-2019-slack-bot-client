### Opis modula
Welcome Bot je sistem koji pruža korisniku opciju da instalira bot u Slack sistem.
Bot je odgovoran za objavljivanje poruka na kanalu ili privatnih poruka.Bot je kontrolisan unutar Bot Admin aplikacije.
### Uloge u sistemu
- Administrator
-Korisnik
